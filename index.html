Sobre o jogo

Sem teto e sem esperança, um portal misterioso abriu diante dele. Do outro lado? Uma dimensão tomada por adolescentes que odeiam pombos.

Imagens do jogo

Como jogar

Use as setas do teclado para se mover como o personagem e o WASD como o pombo.

Use o botão do mouse para atirar pedras como o personagem e o espaço para defecar como o pombo.

Instruções para executar

  1. Baixe o arquivo zip e extraia
  2. Tenha Python instalado (python.org/downloads)
  3. Instale as dependências pelo terminal:
    • pip install pygame
    • pip install opencv-python
    • pip install moviepy
  4. Execute o arquivo principal do jogo no terminal (python jogo.py)
